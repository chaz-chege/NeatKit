Tips for working with colors

Click any format card to copy the value directly to your clipboard — ready to paste into CSS, Figma, Tailwind config, or wherever you need it.
Use HSL for programmatic color manipulation. HSL (Hue, Saturation, Lightness) is the most intuitive format for generating color variations. Keeping hue and saturation fixed while varying lightness produces a consistent shade palette — which is exactly what the palette strip below your color does.
CMYK is for print. Screens use RGB light mixing. Printers use CMYK ink mixing. If you're preparing colors for a print design, check the CMYK values and verify them with your print provider — screen-to-print color matching requires calibration.

Understanding color formats

The same color can be expressed in several formats depending on where you're using it. Here is what each format is for and when to use it.

🔵
HEX (#rrggbb)
Six hexadecimal digits representing red, green, and blue channels (00–FF each). The universal format for web. Used in HTML, CSS, SVG, and most design tools. Three-digit shorthand (#rgb) works when each pair repeats.
🟢
RGB / RGBA
Red, Green, Blue values from 0–255. RGBA adds an alpha channel (0–1) for transparency. Widely supported in CSS. Easier to reason about than HEX when mixing colors mentally.
🌈
HSL / HSV
Hue (0–360°), Saturation (0–100%), Lightness or Value (0–100%). More intuitive than RGB for humans — hue describes the color, saturation its intensity, lightness how bright it is. Great for generating consistent palettes.
🖨️
CMYK
Cyan, Magenta, Yellow, Key (black) — the four ink colors used in print. Unlike RGB (additive light), CMYK is subtractive ink mixing. Use when preparing assets for physical printing or working with a print designer.

Color accessibility — contrast ratios

Web accessibility guidelines (WCAG) require sufficient contrast between text and background colors. The contrast ratio is calculated from the relative luminance of both colors.

✅
WCAG AA — 4.5:1
Minimum contrast ratio for normal text (under 18pt). Required for most accessibility standards. Large text (18pt+ or 14pt bold) only needs 3:1.
⭐
WCAG AAA — 7:1
Enhanced contrast for the highest accessibility level. Recommended for body text in interfaces used by people with low vision. Hard to achieve with coloured text on coloured backgrounds.
⚫
Black on white: 21:1
Maximum possible contrast ratio. Pure black (#000) on pure white (#FFF). Most readable combination. Off-blacks like #111 or #0f0f12 are nearly as readable and feel less harsh.
⚠️
Common failures
Light grey text on white backgrounds. Yellow or light green text on white. Any pastel text on a white or light background. These frequently fail WCAG AA even when they look fine on a designer's calibrated monitor.

Frequently asked questions

Screens use additive RGB light mixing — they emit red, green, and blue light. Printers use subtractive CMYK ink mixing — they absorb light. The color gamut (range of reproducible colors) differs between the two. Some bright RGB colors (like saturated greens and blues) cannot be reproduced in CMYK print and will shift to a duller equivalent. Always do a physical proof print before final production runs.
Both describe color using Hue and Saturation, but differ in the third component. HSL (Lightness) defines 100% as pure white — a fully saturated color at 50% lightness is the purest form of that hue. HSV (Value) defines 100% as the brightest form of the color — 100% value with 100% saturation gives you the pure hue, while 0% value is always black. HSL is more common in CSS; HSV is used in many color picker interfaces.
Use RGBA in CSS: rgba(200, 240, 74, 0.5) where the last value is alpha from 0 (fully transparent) to 1 (fully opaque). In modern CSS you can also use 8-digit HEX: #c8f04a80 where the last two digits are the alpha (00=transparent, FF=opaque). The RGBA value shown in this tool always uses alpha=1 (fully opaque) — adjust the last value in your CSS for transparency.
The palette strip shows two sets of swatches: lightness variations (10% to 90% lightness at your color's hue and saturation), and harmonic colors (complementary at 180°, triadic at 120° and 240°). Click any swatch to load that color into the converter. This is useful for building a consistent color system from a single brand color.
