Tips before compressing

Use Smart Scan Mode for forms, IDs and signed documents. If your PDF is a scanned government form, a filled application, or a document with handwriting and stamps, Smart Scan Mode analyses each page individually and compresses much more aggressively than General mode.
Enable greyscale for official submissions. Most government and institutional portals don't require colour — a blue ink signature and a red stamp are equally legible in greyscale. Enabling this option typically cuts file size by an additional 40–60%.
Use Target Size for portal uploads. If the portal says "maximum 1.91MB", type that in the Target Size field. The tool calculates the compression level needed to hit it rather than you manually retrying at different quality settings.

How it works

This tool compresses PDFs entirely in your browser — your file is never uploaded to any server. It uses PDF.js to render each page and pdf-lib to rebuild the compressed output.

🔍
Per-page analysis
Each page is rendered and analysed for colour variance and content type before compression is applied. Text-heavy pages and image-heavy pages get different treatment.
⚡
Smart Scan Mode
Built specifically for scanned forms, IDs, contracts and signed documents — the exact type of PDF that generic compressors handle worst. Analyses ink density and content before compressing.
🎯
Target size mode
Specify the maximum file size you need and the tool works backwards — calculating the compression level required to hit that target without you having to retry manually.
🔒
100% private
No uploads. No server. PDF.js and pdf-lib run entirely in your browser. Your document — including any ID numbers, signatures, or personal information — never leaves your device.

Why scanned PDFs are hard to compress

A blank form and a filled-in, signed, and stamped version of the same form can differ in file size by 10× or more. Here's why — and why Smart Scan Mode exists.

📝
Handwriting is high-frequency detail
Ink strokes are irregular, high-contrast, and unpredictable. Generic JPEG compression has to preserve all of that detail or the result becomes blurry and illegible — so it can't compress aggressively.
🔴
Stamps are the worst case
Official stamps are dense, colourful, and irregular. A single ink stamp on an otherwise blank page can add hundreds of KB because generic compression has to preserve its full colour and detail.
🎨
Colour takes three times the data
A colour scan stores red, green, and blue channels for every pixel. A greyscale version stores one. For most official documents, the colour information carries no additional meaning.
🏛️
Portal limits are strict
Government and institutional portals set hard limits — 1MB, 2MB, 5MB — that don't account for how a filled-in scanned form actually behaves. Smart Scan Mode is built for exactly this problem.
