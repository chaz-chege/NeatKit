Tips for getting the right size

Keep aspect ratio locked unless you have a specific reason to distort. Unlocking and setting different proportions stretches the image — useful for forcing a cover photo into a banner slot, not for portraits.
Use presets when possible. Platform specs change — 1080×1080 is correct for Instagram feed today, but Stories and Reels have different requirements. Use the presets as a starting point and verify with the platform's current guide.
Downscale, don't upscale. Making an image larger than its original dimensions creates pixels the tool has to invent — results are always blurrier than the original. Always start from the highest-resolution version you have.

How the image resizer works

This tool resizes images entirely in your browser using the HTML5 Canvas API. You set the target dimensions — by typing exact pixel values, picking a common preset, or dragging the percentage slider — and the tool redraws your image onto a canvas at the new size, then exports it as a downloadable file.

The aspect ratio lock works by calculating the ratio of your original width to height. When you change one dimension, the other updates automatically using that ratio. Unlocking it lets you set both dimensions freely — useful for cropping to a specific format like square or widescreen without caring about the original proportions.

Frequently asked questions

Downscaling (making smaller) looks good — you're just using fewer pixels, and the browser averages them intelligently. Upscaling (making larger than the original) always reduces sharpness because the tool has to invent pixels that don't exist. Start from the largest version of your image and scale down.
When you unlock and set a different width-to-height ratio than the original, the image is stretched to fill the new dimensions. This is intentional — sometimes you need to force an image into a specific slot like a banner or avatar that has fixed proportions different from your photo. If you see distortion you didn't want, re-lock the aspect ratio and adjust from there.
The tool accepts images up to 30MB. Very large images above 8000×8000 pixels may take a moment to process on lower-end devices but will complete correctly. Most phone and camera photos process instantly.
The tool accepts GIF files but only processes the first frame — animated GIFs lose their animation and are saved as a static image. For resizing animated GIFs you need a dedicated tool that handles frame-by-frame processing.
