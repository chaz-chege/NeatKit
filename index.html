Common questions
Are these tools really free?
Yes — completely free, forever. No hidden plans, no file size limits, no daily quotas. We keep the lights on with non-intrusive display ads.
Do my files get uploaded to a server?
No. Every tool on Neatkit runs entirely in your browser using JavaScript and WebAssembly. Your images, PDFs, and text never leave your device. This is better for your privacy and makes the tools faster too.
How does browser-side PDF merging work?
We use pdf-lib, a pure JavaScript library that can read and write PDF files entirely within the browser. It's the same approach used by major productivity apps. Quality is identical to server-side processing for most documents.
Can I use these tools offline?
Once a tool page has loaded, most tools will continue to work without an internet connection. We're adding full PWA (offline) support progressively.
Is Neatkit open source?
Yes. The full source code is available on GitHub under the MIT license. Contributions, bug reports, and feature requests are welcome.
